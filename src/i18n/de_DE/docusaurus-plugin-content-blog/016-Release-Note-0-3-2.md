## 更新说明

1. 添加了项目模板，现在可以通过简单的两行命令就创建出所有的项目脚手架

   ```shell
   dotnet new --install Newbe.Claptrap.Template
   dotnet new newbe.claptrap --name HelloClaptrap
   ```

2. 发布了第一篇项目入门指南《第一步——创建项目，实现简易购物车》
